# Android KMP Skills Hub

A curated directory of AI agent skill repositories for Android & Kotlin — covering official and community-built [SKILL.md](https://agentskills.io) repos in one place.

[![Live Site](https://img.shields.io/badge/Live-rohit--554.github.io-7F77DD?style=flat-square)](https://rohit-554.github.io/Android-KMP-Skills-Hub/)
[![License: MIT](https://img.shields.io/badge/License-MIT-1D9E75?style=flat-square)](LICENSE)

---

## What's inside

| Category | Description |
|---|---|
| **Official** | Google, JetBrains, and VoltAgent published skill repositories |
| **Architecture** | Clean Architecture, MVVM, Hilt, modularization |
| **Compose / UI** | Jetpack Compose, Material 3, animations, theming |
| **KMP / CMP** | Kotlin Multiplatform, Compose Multiplatform, navigation |
| **Discovery Hubs** | Curated awesome-lists and skill indexes |

---

## Contributing

To add a repository, open a pull request with a new `<article class="card">` block inside `index.html`. Please ensure the repository contains a valid `SKILL.md` file before submitting.
